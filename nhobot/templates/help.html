Help Page

(This is page is about the upcoming Newbie Slackbot)

What is it?

Newbie is a Slackbot that helps new hires ease into life at Mozilla via timed Slack messages about topics essential to Mozilla employees.

How it works

Newbie will send new hires links to information about topics such as benefits, Workday goals, travel policies, password protection, and All Hands. Messages will be sent to new hires once per day – excluding opt-out, feedback, and help messages – over the course of the new hire's first 30 days. At any time, the new hire can reach this Help page by sending "/newbiehelp help" to Newbie.

Want to opt-out?

Newbie will send an opt-out message to new hires once a week. Select "Stop messages" button at the bottom of the opt-out message and the bot will stop sending the new hire messages.

New hires can also send "/newbiehelp opt-out" to Newbie, and they will stop receiving messages.

Need to file a bug?

Please file a ticket if you experience any issues with Newbie bot, please file a ticket on Jira.

Phase 1 POC Scope

Timeline

Q4 2018

Functionality

For the Phase 1 POC, the Slackbot will push messages out to the user. The user cannot send questions or requests to the bot, except to ask for help.

Content

For the Phase 1 POC, content will be restricted to essential, procedural content essential to employees.

Users

  • United States and Canada-based
  • Full-time employees
